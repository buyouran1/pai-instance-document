```
yum install python-certbot-nginx -y
sed -i "s/renew/renew --nginx/g" /usr/lib/systemd/system/certbot-renew.service
systemctl daemon-reload
systemctl start certbot-renew.service
```

该问题已修复，2019.10.14日以后的用户不会有该问题
